# LINKED LISTS

## MEMORY
1. `NEXT` = link to next node = N
2. `DATA` = data = N
3. `HEAD` = head of list = 1
4. `TAIL` = tail of list = 1 

### TOTAL

\(2N + 1\)

## TESTING METHOD

1. build with docker

```bash
docker build -t list-test .
```

2. Execute tests

```bash
docker run -t list-test
```
